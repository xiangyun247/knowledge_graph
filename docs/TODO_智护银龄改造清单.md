# 智护银龄 —— 原型改造 TODO 清单

**目标**：将现有「胰腺炎 / 胰明」项目改造为「智护银龄 —— 基于 AI 与认知负荷的老龄认知障碍人群低认知负荷知识辅助」原型，**短期内不影响核心功能、保持稳定运行，且符合大创主题**。

**原则**：
- **品牌与文案**：胰腺炎 → 老年认知障碍 / 认知照护；胰明 → 忆路康；系统名 → 智护银龄 / 忆路康。
- **示例与默认值**：仅改展示用示例问题、占位符、提示词中的领域用语，不改变接口与核心逻辑。
- **数据与评测**：eval/init 等数据文件可保留或新建「老年认知」版本，避免破坏现有评测复现性。
